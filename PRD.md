# ManyPaster for Pixso — Product Requirements Document

## Обзор

**Название:** ManyPaster  
**Платформа:** Pixso  
**Тип:** Plugin  
**Аналог:** Many Paster для Figma  

**Назначение:** Массовая вставка текста во множество текстовых слоёв. Позволяет быстро заполнять UI-макеты реальным контентом вместо placeholder-текста.

---

## Функциональные требования

### 1. Основной интерфейс

#### 1.1 Текстовое поле (textarea)
- Многострочное текстовое поле для ввода/вставки текста
- **Разделитель элементов: перенос строки (`\n`)**
- Каждая строка = отдельный элемент для вставки в один текстовый слой
- Содержимое строки может быть любым: длинный текст, знаки препинания, пробелы, спецсимволы — всё это вставляется целиком
- Поддержка copy/paste из внешних источников (Excel, Google Sheets, текстовые файлы)

**Пример:**
```
Привет, мир! Как дела?
Это вторая строка — она пойдёт во второй слой.
А это третья: с запятыми, точками... и прочим!
```
↓ Результат: 3 элемента для вставки в 3 текстовых слоя.

#### 1.2 Кнопка «Вставить» (Paste)
- Вставляет текст из textarea в выделенные текстовые слои
- Порядок вставки: по умолчанию сверху вниз, слева направо (по позиции на canvas)
- Строка 1 → первый выделенный слой, строка 2 → второй, и т.д.

#### 1.3 Кнопка «Скопировать выделенное» (Copy Selected)
- Читает текст из всех выделенных текстовых слоёв
- Вставляет в textarea плагина (каждый слой = отдельная строка)
- Порядок: по позиции на canvas (сверху вниз, слева направо)

---

### 2. Опции (чекбоксы)

#### 2.1 Loop Paste
- **Выкл (default):** Если строк меньше, чем выделенных слоёв — лишние слои остаются без изменений
- **Вкл:** Если строк меньше, чем слоёв — текст вставляется циклически (после последней строки начинается с первой)

**Пример (Loop ON):**
```
Текст: ["А", "Б", "В"]
Слои: 5 штук
Результат: А, Б, В, А, Б
```

#### 2.2 Reverse Paste
- **Выкл (default):** Вставка в прямом порядке
- **Вкл:** Вставка в обратном порядке (последняя строка → первый слой)
- Может комбинироваться с Loop Paste

**Пример (Reverse ON):**
```
Текст: ["А", "Б", "В"]
Слои: 3 штуки
Результат: В, Б, А
```

**Пример (Reverse ON + Loop ON):**
```
Текст: ["А", "Б", "В"]
Слои: 5 штук
Результат: В, Б, А, В, Б
```

#### 2.3 Ignore Symbols
- Чекбокс + текстовое поле для указания игнорируемых символов/слов
- При вставке указанные символы/слова удаляются из текста
- Разделитель: запятая или пробел (уточнить)

**Пример:**
```
Текст: "Hello, World!"
Ignore: ",", "!"
Результат: "Hello World"
```

## Нефункциональные требования

### 3. Сортировка слоёв
- Выделенные слои сортируются по позиции:
  1. Сначала по Y (сверху вниз)
  2. При равном Y — по X (слева направо)

### 4. Валидация
- Если нет выделенных текстовых слоёв → показать предупреждение
- Если textarea пустой при нажатии «Вставить» → показать предупреждение

### 5. UI/UX
- Минималистичный интерфейс
- Сохранение состояния опций между сессиями (если API позволяет)
- Компактный размер окна плагина
